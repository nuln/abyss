# Abyss

Abyss 是一个基于 Go 语言开发的高性能、模块化且插件化的文件服务器和存储管理系统。它采用了清晰的架构设计，支持通过插件进行轻松扩展，旨在提供极致的性能和扩展性。

## 功能特性

- **模块化架构**: 核心系统与具体的存储或协议实现解耦。
- **插件系统**: 通过简单的引入即可扩展功能，支持动态加载。

## 快速入门

### 前置条件

- Go 1.25.x 或更高版本

### 编译

编译标准版：

```bash
make
```

编译专业版：

```bash
make pro
```

### 运行

编译完成后，可以直接运行可执行文件：

```bash
# standard
./abyss

# pro
./abyss-pro
```
